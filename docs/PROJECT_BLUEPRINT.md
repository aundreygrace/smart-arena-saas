# Smart Arena Booking SaaS

## Project Overview
Production-grade sports venue booking platform with realtime booking, analytics dashboard, AI recommendation, and multi-role management system.

---

# Roles

## User
- Search arena
- Booking
- Payment
- Review
- Favorite arena

## Venue Owner
- Create arena
- Manage schedule
- View analytics
- Manage booking

## Admin
- Verify venue owner
- Manage platform
- Manage transactions
- View platform analytics

---

# Business Model

Platform receives admin fee from every booking transaction.

Example:
- Arena Price: Rp100.000
- Platform Fee: 5%
- User Pays: Rp105.000
