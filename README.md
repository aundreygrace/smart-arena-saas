# Smart Arena Booking SaaS

Production-grade sports venue booking platform built using Laravel 12, React, Inertia.js, and PostgreSQL.

## Features
- Multi-role system
- Realtime booking
- Analytics dashboard
- AI recommendation
- Payment integration
- Owner verification

## Tech Stack
- Laravel 12
- React
- Inertia.js
- PostgreSQL
- TailwindCSS

## Status
Currently in active development.
